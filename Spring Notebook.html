Learning Notebook – Unix, Bash, and Python

This notebook summarizes core concepts, commands, and tools learned during the Spring 2026 portion of the course. Each section highlights key utilities and a brief explanation of their purpose.

1. Virtual Machines and Unix Fundamentals

Virtual Machine (VM)
A virtualized Linux environment used to run Unix commands and software independently from the local operating system.

Terminal / Shell
The command‑line interface used to interact with the operating system.

pwd
Prints the current working directory.

ls
Lists files and directories in the current directory.

cd
Changes the current directory.

man
Displays the manual page for a command.

2. Interacting with the VM Using VS Code

VS Code Remote SSH
Allows editing files and running commands on a remote VM through VS Code.

Integrated Terminal
Runs shell commands directly inside VS Code.

File Explorer Panel
Provides a graphical view of files and directories on the VM.

3. Data Wrangling with Core Unix Tools

cat
Displays the contents of a file.

less
Views file contents one page at a time.

head / tail
Displays the first or last lines of a file.

wc
Counts lines, words, and characters in a file.

sort
Sorts lines of text.

uniq
Filters or counts repeated lines.

cut
Extracts specific columns from text.

4. Regular Expressions, grep, and sed

Regular expressions (regex)
Patterns used to match text.

grep
Searches for patterns in files.

grep -v
Excludes matching lines.

grep -E
Enables extended regular expressions.

sed
Stream editor for text transformation.

sed ‘s/pattern/replacement/’
Replaces text within a file or stream.

5. Program and Environment Management

apt
System‑level package manager for Linux.

apt install
Installs software system‑wide.

conda
Environment and package manager for Python and other software.

conda create
Creates isolated environments.

conda activate
Activates a conda environment.

6. Awk

awk
Pattern scanning and text processing language.

$1, $2, …
Refers to columns in text files.

BEGIN / END
Defines actions before or after processing input.

awk ‘{print $1}’
Prints the first column.

7. Bash Scripting

#!/bin/bash
Shebang line specifying the shell interpreter.

Variables
Store values for reuse.

for loops
Iterate over lists or ranges.

if statements
Conditional execution.

chmod +x
Makes a script executable.

8. Integrating Tools and Building Pipelines

Pipes (|)
Passes output of one command as input to another.

Redirection (>, >>, <)
Controls input and output streams.

Command chaining
Combines multiple tools to build workflows.

Bash pipelines
Used to automate text processing and data analysis tasks.

9. Duke Computing Cluster (DCC)

SLURM
Job scheduler used on the DCC.

sbatch
Submits batch jobs.

squeue
Checks job status.

scancel
Cancels jobs.

Globus
Tool for transferring large datasets.

python

print()

import

help()
Displays documentation.

int, float, str

list

tuple

dict

set
Unordered collection of unique elements.

if / elif / else
Conditional statements.

for loops

while loops

break / continue
Control loop behavior.

Matplotlib
